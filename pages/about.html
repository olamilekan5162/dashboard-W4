S/N Tag Name Description Representation
1 Header The header tag defines content that should be considered the introductory information of a page or section header
2 Nav The navigation tag is used for navigation links. It can be nested within the header tag,but secondary navigation nav tags are also commonly used elsewhere on the page. nav
3 Aside An aside element defines content that’s less important. It’s often used for sidebars—areas that add complementary but nonessential information. aside
4 Main This tag contains the main content (also called the body) of a page. There should be only one tag per page. main
5 Section Using section is a way of grouping nearby content of a similar theme. A section tag differs from an article tag. It isn’t necessarily self-contained, but it forms part of something else.
6 Footer You use footer at the bottom of a page. It usually includes contact information, copyright information, and some site navigation. footer
